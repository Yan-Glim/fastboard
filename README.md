# 黑板 · fastboard

基于 Three.js 的在线教学黑板:2D 粉笔绘图、3D 几何体(斜二测投影)、函数图像、AI 框选识别。

## 功能一览

### ✏️ 2D 绘图

| 工具 | 说明 |
|---|---|
| 画笔 (P) | 自由手写,支持颜色/粗细/线型(实线/虚线/点线) |
| 橡皮 (E) | 像素级擦除,只擦 2D,不会误删 3D 多面体 |
| 选择 (V) | 点选 2D 图形或 3D 多面体;拖动移动、角点缩放、旋转;多边形类支持逐顶点改形 |

**图形库(二级菜单,18 种)**

- **线条**:直线、箭头、箭头形
- **多边形**:矩形、三角形、直角三角形、等腰三角形、平行四边形、梯形、菱形、五角星、十字形、多边形(边数 3–20 可调)
- **曲线**:椭圆、半圆、扇形、圆弧、圆环(支持填充)

### 🧊 3D 几何体(16 种)

按类别分组:**多面体**(立方体/四面体/八面体/十二面体/二十面体)、**棱柱**(三/五/六棱柱)、**棱锥**(三/四/五棱锥)、**旋转体**(圆柱/圆锥/圆台/球)。

- 只渲染棱线:正面实线、背面虚线;球体为经纬线网格
- 编辑层级:**1** 对象 / **2** 顶点 / **3** 边;拖动顶点或整条边改形
- **旋转**:🔄 按钮开启自由旋转,或 Shift+拖动按 45° 步进;绕自身 X/Y/Z 轴,顶点级绕选中顶点、边级绕选中边
- **✂ 切割**:点选棱上或曲面上的点(每条棱最多保留最新 2 个,点已选点可移除),≥3 个点出现闪烁虚线预览,再点切割按钮或按 **Enter** 执行,**Esc** 取消
- 滚轮缩放选中物体

### 📈 函数图像

- 可开多个浮动窗口,每个窗口可画多条曲线(颜色自动区分)
- 表达式支持 `x^2`、`sin(x)`、`cos(x)`、`tan(x)`、`log(x)`、`ln(x)`、`sqrt(x)`、`abs(x)`、`pi`、`e` 等
- XY 轴范围手动控制(Xmin/Xmax/Ymin/Ymax),也可勾选"自动Y";滚轮以光标为中心缩放 X 范围
- 窗口可拖动、可缩放

### 🤖 AI 框选识别 (A)

框选黑板区域,VL 模型结构化识别并自动分发:

| 识别结果 | 动作 |
|---|---|
| 数学公式 | 加入函数窗口画曲线 |
| 2D 图形 | 在框选区域原位创建对应粉笔图形(可继续编辑) |
| 3D 图形 | 在框选中心创建对应立体,尺寸跟随框选大小 |

### 通用操作

- **Ctrl+Z** 撤销 / **Ctrl+Y** 重做(2D/3D 统一历史栈,共 100 步)
- **Delete** 删除选中对象
- 清空按钮清除整个黑板(需确认)

## 启动服务

### 1. 配置(仅首次)

```bash
cp config.example.yml config.yml
# 编辑 config.yml,填入 VL 模型配置:
#   base_url: https://你的-vl-接口地址/v1
#   api_key: 你的密钥
#   model: 模型名
```

> `api_key` 只存在服务端,前端永远接触不到;`config.yml` 已被服务器禁止 HTTP 访问。

### 2. 启动

```bash
python server.py          # 默认 8000 端口
python server.py 6722     # 指定端口
```

浏览器打开 `http://localhost:8000`(或对应端口)。

### 3. 后台运行(服务器部署)

```bash
nohup python server.py 6722 > server.log 2>&1 &

# 管理
ps aux | grep server.py    # 查看进程
tail -f server.log         # 查看日志
pkill -f "python server.py" # 停止
```

防火墙/安全组需放行对应端口,然后访问 `http://你的公网IP:6722`。

## 部署方式

| 方式 | 说明 |
|---|---|
| **同源部署(默认)** | 前端和 API 都由 `server.py` 提供,前端请求相对路径 `/api/recognize`,无需任何配置,换 IP/域名也不用改代码 |
| **反向代理** | Nginx 托管静态文件,`location /api/ { proxy_pass http://127.0.0.1:6722; }` |
| **前后端分离** | 编辑前端目录下 `api-config.js`,填 `window.BLACKBOARD_API_BASE = 'http://API地址:6722'`;服务端已开放 CORS |

> HTTPS 前端必须配 HTTPS 的 API,否则浏览器拦截混合内容。

## 目录结构

```
├── index.html        # 页面与工具栏
├── style.css         # 样式
├── api-config.js     # 前端 API 地址配置(前后端分离时改这里)
├── server.py         # 静态文件 + /api/recognize 识别接口(多线程)
├── config.yml        # VL 模型密钥(不入库,禁止 HTTP 访问)
├── js/
│   ├── main.js       # 入口:渲染循环、工具切换、指针路由、快捷键
│   ├── chalk.js      # 2D 粉笔层(手写/图形/选择编辑)
│   ├── solids.js     # 3D 几何体层(棱线渲染/旋转/切割)
│   ├── functions.js  # 函数图像窗口
│   └── ai.js         # AI 框选识别
└── vendor/           # Three.js 及线条扩展(本地化,无外部依赖)
```
